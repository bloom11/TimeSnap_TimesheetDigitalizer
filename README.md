An AI-powered Progressive Web App (PWA) that digitizes physical timesheets. 
It uses advanced Vision AI (Gemini, OpenAI, Claude, Mistral) to extract work hours from photos, 
automatically maps the data to your company's Excel templates, and supports cross-device syncing between your phone camera and desktop.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`

2. Run the app:
   `npm run dev`
